?이게 무슨 작업인가요

먼저 무엇을 하게 되는지 편하게 이해하고 시작하세요. 겁먹을 필요 없습니다.

한 줄 요약: Claude Code(AI 코딩 비서)를 시켜서 구글 광고(디멘드젠)를 만드는 작업입니다. 디멘드젠은 유튜브·Gmail·디스커버에 나오는 이미지·영상 광고예요.

방법은 두 가지가 있고, 이 가이드는 둘 다 안내합니다. 어려운 쪽(B)은 AI가 대부분 자동으로 하고, 사람은 로그인·결제·최종 확인만 하면 됩니다.

지금 바로 가능

A · 지금 광고 만들기 (테스트)

AI가 구글애즈 화면을 띄우고 "여기 누르세요"를 하나씩 안내합니다. 오늘 바로 광고를 초안으로 만들어볼 수 있어요. 특별한 승인 필요 없음.

며칠 뒤 완성

B · 대화로 자동 생성 (본격)

"예산 5만원 광고 만들어줘" 한마디로 AI가 자동으로 광고를 만드는 서버를 구축합니다. 구글 승인(최대 5일)이 필요해요.

권장 순서: 먼저 B를 시작해 승인 시계를 걸어두고, 그 기다리는 동안 A를 테스트로 해봅니다. 5일 뒤 승인되면 B를 완성해 본격적으로 씁니다. (아래 타임라인 참고)

▶시작하기 — 딱 한 줄

Claude Code(앱)를 열고, 아래 상자를 복사해서 붙여넣기 하세요. 그게 전부예요.

Claude Code에 붙여넣기 ↓
https://github.com/aiebrain/google-ads-demandgen-kit
이 저장소로 START-HERE.md 대로 진행해줘

그러면 AI가 알아서: 필요한 파일 설치 → B 시작(승인 걸기) → A 테스트 → 5일 뒤 B 완성 순서로 안내합니다.

※ 링크만 붙여넣으면 저절로 실행되진 않아요. 위 두 줄(링크 + 지시)을 함께 넣어야 시작됩니다.

1준비물 (미리 챙기기)

아래만 있으면 됩니다. 대부분 무료예요.

  • 구글 계정(Gmail) 1개 — 처음부터 끝까지 이 계정 하나로만 진행하세요 (섞으면 오류)
  • 신용/체크카드 1장 — 구글 클라우드 결제 등록용. 무료 크레딧 제공, 실제 청구는 거의 없음
  • Claude Code 설치 (AI 비서 앱)
  • Python 3.10 이상 (설치 안내는 AI가 도와줌)

2진행 흐름 (한눈에)

"오늘 셋업하면 오늘 자동광고"는 아니에요. 구글 승인 때문에 며칠 걸리는 대기가 있습니다. 대신 그 사이 A로 먼저 체험합니다.

지금 · B 시작

서버 셋업으로 "승인 시계" 걸기

관리자 계정 → 개발자 토큰 발급 → Basic 신청 제출 → 클라우드 프로젝트·결제 → 로그인 인증. 여기서 두 개의 대기가 시작됩니다.

지금 · A 테스트

UI로 디멘드젠 광고 만들어보기 (초안)

기다리는 동안 AI 안내로 광고를 초안까지 만들어 봅니다. 이건 테스트/데모예요 — 게시 안 함, 돈 안 나감.

약 24시간 뒤

클라우드 결제 활성화

카드 등록 후 결제가 켜지는 데 최대 24시간. 켜지면 서버 배포가 가능해집니다.

약 5영업일 뒤

개발자 토큰 승인(Basic)

구글이 신청을 심사(최대 5영업일). 승인 메일이 오면 실제 계정에 자동 광고 생성이 열립니다.

승인 후 · B 완성

배포 → 등록 → 대화로 자동 생성

AI가 서버 배포·연결을 마치면, 이제 "예산 5만원 디멘드젠 만들어줘" 한마디로 광고가 만들어집니다(항상 일시중지로 안전하게).

즉, 두 개의 대기가 핵심입니다. ① 결제 활성화(≈24시간) ② 개발자 토큰 Basic 승인(≈5영업일). 이 둘은 구글이 처리하는 거라 기다려야 합니다. 그 사이 A로 광고를 미리 체험하세요.

3누가 무엇을 하나 (역할 분담)

AI가 대부분 대신 해줘요. 당신은 사람만 할 수 있는 것만 하면 됩니다.

🤖 AI(Claude)가 하는 것
정확한 페이지 열기 · 화면 읽고 "여기 누르세요" 안내 · 안전한 값 대신 입력 · 클라우드 설정 자동 · 결과 검증
🙋 당신이 직접 하는 것
구글 로그인 · "로봇이 아닙니다"(캡차) · 카드 입력 · 동의/허용 클릭 · 예산·최종 게시 확인
안심 포인트: 모든 광고는 일시중지(초안)로 만들어지고, 실제로 켜는 건 당신이 확인할 때만 됩니다. 비밀번호·카드·토큰은 AI가 입력하지 않아요.

4알아둘 것 (대기 · 브랜드 인증)

📧 소통은 전부 "이메일"로 옵니다

개발자 토큰 신청 접수·추가 질문·최종 승인/거부까지 모두 신청서에 적은 연락처 이메일로 옵니다. 자주 확인하는 주소를 쓰고, 받은편지함을 계속 확인하세요.

🏷️ 브랜드 인증 — 필수 아님(권장)

신청 후 받는 메일에 "브랜드 인증을 하면 심사가 빨라진다"고 나옵니다. 안 해도 정상 심사(≈5영업일)는 진행돼요. 더 빨리 받고 싶을 때만, 본인이 직접 구글 클라우드 콘솔에서:

  • 개발자 토큰을 클라우드 프로젝트에 연결
  • OAuth 앱을 외부 + 프로덕션(게시) 으로 전환
  • 브랜딩 정보(앱 이름·로고·홈페이지·도메인) 입력 → "브랜딩 확인" → "브랜딩 게시"

※ 도메인은 소유 확인이 필요할 수 있어요. 이 작업은 AI가 대신 하지 않고 방법만 안내합니다.

Q자주 묻는 질문

코딩을 전혀 몰라도 되나요?

네. AI가 명령어 실행·설정을 대신 하고, 당신은 로그인·클릭·확인만 합니다.

돈이 나가나요?

셋업 자체는 거의 무료(클라우드 무료 한도)입니다. 광고는 당신이 직접 게시(활성화)할 때만 광고비가 나갑니다. 테스트(A)는 초안이라 0원.

왜 5일이나 기다리나요?

구글이 개발자 토큰을 Basic 등급으로 승인해야 실제 계정에 자동으로 광고를 만들 수 있어요. 그 심사가 최대 5영업일입니다. 그 전엔 A(수동 UI)로 만듭니다.

링크만 붙여넣으면 자동으로 다 되나요?

완전 자동은 아니에요. 링크 + "START-HERE.md 대로 진행해줘" 두 줄을 넣으면 AI가 순서대로 안내를 시작합니다. 중간중간 당신의 로그인·확인이 필요합니다.

다른 컴퓨터에서도 되나요?

됩니다. 같은 서버는 주소+토큰만 있으면 새 PC에서 바로 연결돼요. 다른 계정으로 새로 하려면 위 과정을 그 계정으로 한 번 더 진행하면 됩니다.
