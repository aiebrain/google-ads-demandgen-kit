# UI 코파일럿: 디멘드젠 캠페인 "생성" 실행 명세 (에이전트용)

이 문서는 **Claude(에이전트)가 직접 따라 실행**하는 절차다. **개발자 토큰·MCP 서버 배포·Basic 승인 없이**,
Google Ads 웹 UI를 내장 브라우저로 띄워 **디멘드젠 캠페인을 클릭 단위로 생성**한다(= "A 방식").
사용자가 "지금 디멘드젠 광고 만들어줘 / 스크린샷처럼 UI로 안내해줘 / API 승인 기다리지 말고 지금" 이라고 하면 이 방식을 쓴다.

> ⚠️ 이건 셋업(google-ads-demandgen-setup의 서버 구축)과 **다른 목적**이다. 여기선 서버가 필요 없다.
> 필요한 전제는 단 하나: **광고를 만들 Google Ads 계정에 로그인**돼 있을 것. (실제 게재하려면 그 계정에 결제 카드 필요.)

## 절대 하지 않는 것 (사용자에게 넘김)
- 로그인/비밀번호, reCAPTCHA, **약관·데이터처리 동의 클릭**, **예산 확정 입력(먼저 금액 확인)**,
  **최종 "게시/캠페인 게시"**, 파일 업로드.
- 특히 **게시(활성화)는 사용자 명시 확인 없이는 절대 누르지 않는다.** 전부 **초안/일시중지**로만 만든다.

## 실측 검증된 흐름 (2026-07 실제로 완주)

### 0. 대상 광고계정으로 전환
- 내장 브라우저 `ads.google.com` → MCC면 계정 전환기(상단)로 **하위 광고계정** 선택.
  (신규 계정은 "첫 번째 캠페인 만들기" 온보딩이 뜰 수 있음 → X로 닫고 진행)

### 1. 새 캠페인 → 유형
- "+ 새 캠페인" → 목표 화면에서 **"안내 없이 캠페인 만들기"** → 계속
- 캠페인 유형에서 **"디맨드젠 캠페인"** 선택

### 2. 중점 목표 + 향상된 전환
- 중점 목표: 테스트/무전환추적이면 **"페이지 조회"** 등 가벼운 것. (라디오 클릭이 빗나가기 쉬움 — 확대해 정확히)
- ⚠️ "향상된 전환 사용" 체크가 켜져 있으면 버튼이 **"동의 및 계속하기"**(데이터 처리 약관)로 바뀐다.
  → 이 동의 클릭은 **사용자가**. (체크 해제는 UI가 잘 안 먹힘 → 그냥 사용자가 동의 클릭)

### 3. 캠페인 설정
- **캠페인 목표(입찰)** = **"클릭수"** 선택 → **전환추적 없이도 진행**(전환 액션 경고 사라짐). (전환추적 있으면 "전환")
- 캠페인 이름: 기본값 OK
- **예산**: ⚠️ 지출 직결 → **금액을 먼저 사용자에게 확인**하고 입력. 일시중지라 실지출은 0이지만 필수 입력값.
  (₩5,000 같은 소액도 됨; "예산 낮음" 경고는 권장일 뿐 진행 가능)
- **EU 정치 광고** (필수): **"아니요"** 선택. (API의 `contains_eu_political_advertising`와 동일 요구)

### 4. 광고그룹
- 위치: **대한민국**(신규 계정 기본 선택돼 있는 경우 많음) / 언어: **한국어** 추가
- 채널: **모든 Google 채널**(기본) / 잠재고객: 선택사항(테스트면 기본 최적화)

### 5. 광고 소재 (핵심)
- 유형: **단일 이미지 광고**(가장 간단)
- **최종 URL** 입력(필수)
- ★ **이미지**: 최종 URL을 넣으면 구글이 **그 사이트를 스캔해 "추천 이미지"를 제안**한다.
  **"전체 선택"** 하면 자동 크롭되어 들어감 → **파일 업로드 불필요**(내장 브라우저 업로드 한계 우회!).
- **광고 제목(헤드라인)** 1개+ , **설명** 1개+ 입력(제안 문구 재사용 또는 직접 입력). **업체명** 입력(필수).
- CTA(클릭 유도문안): 자동으로 둠. 로고(1:1)는 선택(넣으면 광고 효력↑).

### 6. 리뷰 → 초안으로만
- "리뷰로 이동" → 요약 확인. 결제 미설정/광고효력 미완료면 **"게시할 수 없습니다"** 로 뜸(정상, 실수 게시 방지).
- **게시하지 않는다.** 좌상단 **X로 나가면 자동으로 "캠페인 초안"으로 저장**된다(계정 개요에 "캠페인 초안 작업 계속"으로 표시).
- 실제 게재를 원하면: 광고계정에 **결제 카드 연결 후**, 사용자 확인을 받고 그때만 게시.

## 요약 규칙
- 입찰 **클릭수**로 전환추적 회피 / EU정치광고 **아니요** / 이미지는 **사이트 스캔 추천 이미지**로 업로드 회피 /
  예산·게시는 **사용자 확인** / 결과는 **초안**. 이 방식은 **API 셋업·Basic 승인과 무관하게 오늘 바로** 된다.
