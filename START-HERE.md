# START-HERE — 이 저장소로 시작하기

사용자가 이 저장소 링크를 Claude Code에 주며 **"START-HERE대로 진행해줘"** 라고 하면,
Claude(에이전트)는 아래 절차를 그대로 따른다. **권장 순서: B 착수 → 대기 중 A(테스트) 즉시 → 5일 뒤 B 완성.**

---

## 시작할 때 사용자에게 먼저 설명할 것 (그대로 안내)

> **진행 방식(권장 순서)**: 먼저 **B(API 자동화 서버 셋업)** 를 진행해 시간이 걸리는 구글 승인 두 개
> — 개발자 토큰 **Basic 승인(최대 5영업일)** 과 **GCP 결제 활성화(최대 24시간)** — **시계를 먼저 걸어둡니다.**
> 그 대기 동안 **A(Google Ads UI로 디멘드젠 광고 만들기)를 "테스트/데모"로 지금 바로** 해봅니다.
>
> ⚠️ **A는 테스트입니다.** B의 "대화 한마디로 자동 생성"과는 다른, **UI를 손수 클릭해 만드는 방식**이고
> 결과는 **초안(미게시)** 입니다. 실제 자동화 광고 생성은 **5일 뒤 Basic 승인이 나면 B를 마저 완성**해서 씁니다.

---

## 0. 설치 (에이전트)
1. 로컬에 없으면 `git clone https://github.com/aiebrain/google-ads-demandgen-kit`
2. `skills/google-ads-direct-mcp` 와 `skills/google-ads-demandgen-setup` 를 `~/.claude/skills/` 로 복사
3. 위 "사용자에게 먼저 설명할 것"을 안내한 뒤 진행

## 1. B 착수 — 서버 셋업으로 "승인 시계" 걸기
`skills/google-ads-demandgen-setup/references/browser-copilot-walkthrough.md` 대로 내장 브라우저 코파일럿으로:
- **Step 1** 관리자(MCC) 계정 생성(서울/KRW)
- **Step 2** 개발자 토큰 발급(Test) → **Basic 액세스 신청서 제출** (설계문서는 `basic-access-design-doc.md`
  템플릿에 값만 채워 첨부) — **여기서 5영업일 승인 시계 시작**
- **Step 3** GCP 프로젝트 + 결제(카드) + Ads API — **결제 활성화 최대 24시간 시계 시작**
- **Step 4** OAuth 동의화면·테스트사용자·데스크톱 클라이언트
- **Step 5** 리프레시 토큰(`python -u oauth_setup.py` → 브라우저 "허용")
- **Step 6** 로컬 self-test로 자격증명 검증
- 🛑 **여기서 멈춘다.** Step 7 배포는 **결제 활성화 후**, 실제 API 광고 생성은 **Basic 승인 후**. 두 대기를 사용자에게 알림.

## 2. A 즉시 — "테스트/데모"로 지금 바로 (대기 동안)
`skills/google-ads-demandgen-setup/references/ui-copilot-demandgen-campaign.md` 대로:
- 광고계정에 디멘드젠 캠페인을 **초안까지** 생성(입찰=클릭수, EU정치광고=아니요, 사이트 스캔 이미지, 예산·게시는 사용자 확인).
- ★ **반드시 명시**: "이건 **테스트/데모**입니다 — B의 자동화가 아니라 UI 수동 방식이고, **초안(미게시)** 이며
  실제 게재는 광고계정 결제 + 게시 확인이 필요합니다. 진짜 자동 생성은 5일 뒤 B 완성 후."
- 테스트가 끝나면 초안은 남기거나 삭제(사용자 선택).

## 3. 5일 뒤 — B 완성 (승인 후 재개) + 이게 무슨 내용인지 설명
**언제**: 개발자 토큰 **Basic 승인 메일** 도착(+ GCP 결제 활성화 완료) 후.
**왜 5일**: 발급 직후 토큰은 **Test 등급**이라 테스트 계정만 호출 가능. **Basic 승인**이 나야 **실계정에 API로
광고 생성**이 허용된다. 그래서 그전엔 A(UI 수동)만 되고, B(API 자동)는 이 승인을 기다린다.
**무엇을 하나 (재개 절차)**:
1. **Cloud Run 배포** — `deploy_cloud_run.py`(또는 `bootstrap.py`)가 시크릿 생성·IAM·컨테이너 빌드·배포까지 원샷. (결제 활성화 후에만 성공)
2. **Claude Code 등록** — `claude mcp add ...` 로 MCP URL + 베어러 토큰 연결.
3. **검증** — `verify_demandgen.py` 로 "디멘드젠 준비 완료 ✅".
4. **실사용** — 이제 **대화 한마디로**("예산 5만원 디멘드젠 만들어줘") API가 광고를 생성(전부 PAUSED, 활성화는 확인 인자).
- 사용자에게: "Basic 승인 메일 오면 알려주세요 → 배포부터 자동으로 이어가 B를 완성합니다."
